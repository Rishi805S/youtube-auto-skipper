SponsorSkip Extension Testing

Step 1: Load Extension in Chrome

  1. Open Chrome and go to chrome://extensions/
  2. Enable "Developer mode" (toggle in top right)
  3. Click "Load unpacked"
  4. Select the dist folder from this project

Step 2: Test on YouTube

  1. Go to any YouTube video with sponsor segments
  2. Open browser console (F12 â†’ Console tab)
  3. Look for logs starting with [SponsorSkip]
  4. Click the extension icon to open popup

Step 3: Expected Behavior

  • Console should show: [SponsorSkip] Simple content script loaded
  • When segments are found: [SponsorSkip] Found segments: [...]
  • When skipping: [SponsorSkip] Skipping segment: {...}
  • Toast notifications should appear when segments are skipped

Step 4: Test Videos with Sponsors

Try these types of videos that commonly have sponsor segments:

  • Tech review videos
  • Gaming videos
  • Educational content

Troubleshooting

If not working, check:

  • Extension is enabled in chrome://extensions/
  • Console for any error messages
  • Network tab for SponsorBlock API calls
  • Popup toggle is enabled
